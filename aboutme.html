Reflection Week 2.2

Python and Scratch both seem to be programming languages with the same concepts but Python is definitely harder in most aspects. Scratch has a GUI structure that is more dragging and dropping and fitting things together. If you understand basic programming concepts, you should be good with Scratch. Python is more difficult because it requires exact syntax and knowing how to use a text editor to run the program. There's a lot less given to you with Python. It was different today because it was more of a lecture oriented type of learning than a project based learning. There was more theory and concepts to learn Than actual doing. I don't like it as much but I do understand it is necessary. Programming in Python is definitely harder because the syntax messes you up so much but I like that we had more flexibility. It was also cool to really figure out how to spiral a shape properly with user input.
